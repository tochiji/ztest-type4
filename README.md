# ztest-type4
母比率の差の検定（タイプ4）を行う小さなプログラムです

## 使用法
usage: ztest-type4.py 引数1 引数2 引数3 引数4

このプログラムは、4つの引数が必要です。  
 1. 全体のn数
 1. 全体における比率p
 1. ある属性のn数
 1. ある属性における比率p1

### 例  
~~~
$ python3 ztest-type4.py 150 0.3 100 0.45
1%有意 (検定統計量:5.66947)
~~~

ただし、
 - それぞれn数は30以上かつ比率pは[0<=p<=1]を満たすこと  
 - 「1.全体のn数」は「3.ある属性のn数」より大きいこと
